ANTES DE TODO ESTO NOS DIO UN PROBLEMA QUE SE RESOLVIO DE LA SIGUIENTE MANERA EN VISUAL STUDIO CODE:
El error 401 (Unauthorized) indica que no tienes autenticación configurada para acceder al feed privado de Azure DevOps NuGet. Vamos a revisar la configuración y solucionar el problema.

Se buscaron archivos que coinciden con **/nuget.config, no hay coincidencias

Se ha buscado texto pkgs.dev.azure.com, sin resultados

No hay un archivo nuget.config local en el proyecto. El feed de Azure DevOps probablemente está configurado a nivel global o en los archivos .csproj. Vamos a verificar:


dotnet nuget list source
 Enable shell integration to improve command detection

iex "& { $(irm https://aka.ms/install-artifacts-credprovider.ps1) }"
